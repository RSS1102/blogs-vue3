
1. ##### 使用`hover`控制 子、父，兄 组件样式。


   1. 父控制子

      ```vue
      <div class="a">
        <div class="b">b</div>  
      </div>
      
      <style>
      .a:hover b{
      background-color:red;
      }
      </style>
      ```

      

   2. 子控制父。

      ```vue
      <div class="a">
        <div class="b">b</div>  
      </div>
      
      <style>
      .a:has(.b:hover) {
      background-color:red;
      }
      </style>
      ```

      

   3. 兄弟组件(同阶元素)。

      ```vue
      <div class="c">
         <div class="a">a</div>  
         <div class="b">b</div>  
      </div>
      
      <style>
      .a:hover~b {
      background-color:red;
      }
      </style>
      ```

      

2. ##### `grid`布局

   1. 使用`grid`进行layout布局

      ```vue
      <div class="main">
        <div class="a"></div>
        <div class="b"></div>
      </div>
      <style>
          .main {
            display: grid;
            grid-template-rows: 1fr auto;
          }
          .a{
              hight:300px;
          }
        // b 会铺满剩下的位置  
      </style>
      ```
