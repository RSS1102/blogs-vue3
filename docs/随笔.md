
1. ##### 使用`hover`控制 子、父，兄 组件样式。


   1. 

      ```html
      <div  class="a">
        <div class="b">b</div>  
      </div>
      
      .a:hover b{
      background-color:red;
      }
      ```

   2. 子控制父。

      ```html
      <div  class="a">
        <div class="b">b</div>  
      </div>
      
      .a:has(.b:hover) {
      background-color:red;
      }
      ```

      3.兄弟组件(同阶元素)。

      ```html
      <div  class="c">
         <div class="a">a</div>  
         <div class="b">b</div>  
      </div>
      
      .a:hover~b {
      background-color:red;
      }
      ```
